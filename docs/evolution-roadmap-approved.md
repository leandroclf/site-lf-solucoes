# Roadmap de Evolucao Aprovado

Este documento registra a sequencia de evolucao aprovada para o site institucional.

## Fase A - Guardrails de qualidade

1. Baseline tecnico e metas
2. Lighthouse CI em observacao
3. Lighthouse CI com gate bloqueante

## Fase B - UX/UI institucional

1. Design system minimo (tokens e componentes base)
2. Navegacao e arquitetura de informacao consistentes
3. Microinteracoes leves com preferencia por CSS

## Fase C - Interatividade de negocio

1. Modulo Diagnostico 60s (local, sem backend)
2. Modulo Simulador ROI (faixas conservador/provavel/agressivo)

## Fase D - Opcional e controlada

1. Spike de viabilidade Astro sem cutover
2. IA via endpoint serverless externo (com fallback local)
3. Hardening final (SEO/A11y/performance)
